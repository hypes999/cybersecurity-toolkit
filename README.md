# Cybersecurity Toolkit

Conjunto de ferramentas simples em Python para uso em cibersegurança: geração de passphrases, pequenas análises de segurança e automação do dia a dia.  
O objetivo deste repositório é concentrar num único sítio os scripts e pequenas apps que realmente uso e que podem ser úteis para outras pessoas.

## Como usar

1. Clonar o repositório.
2. Entrar na pasta do projeto.
3. Escolher a ferramenta pretendida e seguir as instruções do respetivo `README.md`.

## Objetivo

Este projeto serve como portefólio técnico em cibersegurança e como coleção de ferramentas que uso para aprender, automatizar tarefas e aplicar boas práticas de desenvolvimento seguro.  
Contribuições (issues, sugestões ou pull requests) são bem‑vindas.
